Análisis de rendimiento de neumaticos.

Este proyecto analiza el rendimiento y la durabilidad de distintas marcas de llantas utilizadas en el transporte público de la ruta Tlaxcala–Apizaco mediante herramientas de análisis estadístico. El objetivo principal fue identificar qué alternativas ofrecen un mejor desempeño en términos de kilometraje y costo por kilómetro, con la finalidad de apoyar la toma de decisiones de compra para los operadores de transporte público. El análisis se realizó utilizando técnicas estadísticas como ANOVA, permitiendo comparar el comportamiento de diferentes marcas y determinar la variable con mayor relevancia para la evaluación del rendimiento de las llantas.

Tecnologías usadas

Python Excel
